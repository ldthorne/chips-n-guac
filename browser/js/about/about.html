This website was built by Daniel Thorne with the express purpose of allowing anyone to get a coupon for BOGO burritos from Chipotle without playing their online game. The repository for this website can be seen here.


This web application was built using Joe Alves' amazing generator, FSG.


This application was built in response to the discovery that Chipotle's game, Guac Hunter, was making a simple, insecure, unencrypted request to their server to indicate a user's completion of the game, which prompted a text with a coupon to be sent to the user. This application simply collects the user's information, concatenates it, and sends a request to Chipotle's server.


I am collecting user's first name so that I may accurately track the number of users.
